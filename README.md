# Food Delivery Performance Dashboard 📊

An end-to-end data analytics command center built in Power BI to track platform performance, optimize delivery windows, and analyze regional consumer cuisine preferences.

## ⚡ Core Architecture & Highlights
* **Relational Data Model:** Engineered a dynamic schema linking `Cities` ➔ `Restaurants` ➔ `Orders` tables with full bi-directional cross-filtering capabilities.
* **Chronological Optimization:** Wrote custom DAX expressions to untangle and re-sort transactional timestamps into a clean, 24-hour daily operational workflow.
* **Minimalist UI/UX:** Styled with a sleek, low-distraction layout maximizing whitespace and high-contrast typography for immediate analytical clarity.

## 🛠️ Technology Stack
* **Database Engine:** SQL (Data retrieval & manipulation)
* **BI Platform:** Microsoft Power BI Desktop
* **Language:** DAX (Data Analysis Expressions)

## 📈 Key Metrics Tracked
* **Total Revenue:** $8.18M macro platform spend
* **Average Order Value (AOV):** Dynamic calculation of basket size per transaction
* **Hourly Volatility:** Identifying peak demand spikes to streamline delivery partner allocation
